Detailed Description

A class to lower and raise the shooter so that it's easier to hit higher targets without being blocked.

Methods:

  • IsAtTop() returns true if the lift is at the very top
  • IsAtBottom() returns true if the lift is at the very bottom
  • Raise() simply raises the elevator by some increment (unless it's at top). You must call 'Raise' repeatedly in a loop to actually lift the elevator.
  • Lower() the same thing, except to lower the lift
  • Raise(float magnitude) manually set the speed controller to go up by some speed
  • Lower(float magnitude) Same thing, except to lower.
  • Stop() Forcibly stop the elevator
