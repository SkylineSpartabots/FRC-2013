Detailed Description

Aims the turret at the given target.

Protip: If you set this to the default turret command, using 

   Tracking::ClosestOffset

...as the desiredTarget param, the turret will automatically lock on and track the target it's pointing at.

You can manually adjust the target to point at a different target, or issue a command to point it at the appropriate target and let this take over once that command is finished.
